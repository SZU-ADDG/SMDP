## [LRN-20260712-001] best_practice

**Logged**: 2026-07-12T14:00:00+08:00
**Priority**: medium
**Status**: applied
**Area**: docs

### Summary

Keep public experiment documentation aligned with both the accepted manuscript and the files actually included in the release.

### Details

The paper provides authoritative method, protocol, result, author, and funding information. The code package remains authoritative for available entry points, data files, and editable presets. README claims should distinguish reported paper results from repository availability.

### Suggested Action

Cross-check future README updates against the manuscript, current directory tree, and final IEEE bibliographic record.
